# TimeLogger v0.1
## A Spigot Plugin to check player's playtime. 

## Commands
* `/playtime [player] <time>
  - Checks playtime of a player. If no time is specified, it will check all playtime.
  - Valid time units: seconds (s), minutes (m), hours (h), days (d).
  
## Permissions
* `timelogger.playtime`
